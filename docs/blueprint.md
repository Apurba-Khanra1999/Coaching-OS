# **App Name**: TuitionFlow

## Core Features:

- Integrated Dashboard Analytics: Visual summary cards for students, attendance, and fees, featuring growth charts and recent activity logs.
- Student Lifecycle CRM: Comprehensive database for student profiles, admission details, and parent contacts with CSV export.
- Dynamic Batch Management: Tools to organize subjects, teachers, and timings with student assignment capabilities.
- Automated Attendance Engine: Batch-wise daily attendance marking with bulk actions and historical reporting.
- Advanced Fee Ledger: Tracking of billing cycles, partial payments, and digital receipt generation with balance calculation.
- Exam & Gradebook Suite: Modular exam design, score entry forms, and automated generation of rankings and report cards.
- Parent Communication Center: Targeted announcements and notifications for attendance, fees, and exams using predefined templates.
- Centralized Settings Portal: Configuration for institute profiles, logo management, academic sessions, and system-wide preferences.

## Style Guidelines:

- Primary color: A deep, professional Indigo (#4F46E5) to evoke stability and trust in educational management.
- Background color: A refined, very light Gray-Blue (#F9FAFB) that maintains high contrast for clarity.
- Accent color: An elegant Violet (#7C3AED) used for interactive states and highlight icons.
- Font pairing: 'Space Grotesk' for modern, tech-forward headlines and 'Inter' for highly readable body text.
- Minimalist, stroke-based line icons to maintain a lightweight SaaS aesthetic across the sidebar and cards.
- Permanent sidebar navigation with a responsive modular grid 'Bento Box' layout for dashboard widgets.
- Subtle slide-and-fade transitions when switching between module views to provide a fluid, app-like experience.
